# Git Workflow del Proyecto

## Comandos básicos

- git status  
  Ver el estado real del proyecto.

- git add  
  Seleccionar cambios para el commit.

- git commit  
  Registrar una decisión.

- git push  
  Publicar cambios.

## Reglas personales

- Guardar archivos antes de commitear.
- Revisar git status antes de actuar.
